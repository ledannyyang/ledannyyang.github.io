Tutor: Pere-Pau Vázquez Alcocer

Bienvenido a la página web del TFM, se ha creado con el propósito de mostrar la Demo de la aplicación. En la parte superior derecha se encuentra un link para descargar el documento PDF asociado al proyecto de TFM. Espero que os guste y os sea de utilidad.

En este vídeo se muestra de manera breve el funcionamiento de la aplicación. Se puede observar la lista de conversaciones que posee el usuario, el perfil de usuario así como la acción de actualizar la imagen de perfil. Posteriormente se descarga el modelo de traducción, en este caso, el castellano.

En este vídeo se muestra una conversación ficticia entre dos usuarios, un usuario castellano hablante y el otro usuario cuya lengua es el mandarín. Como se puede observar en el vídeo, ambos usuarios pueden mantener una conversación de manera correcta.

En este vídeo, el usuario muestra la funcionalidad de enviar una imagen guardada en la memoria de su dispositivo y un GIF (Graphic Interchange Format)
